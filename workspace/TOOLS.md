# TOOLS.md - 本地笔记

技能定义了工具_如何_工作。这个文件是记录_你的_具体情况——那些独特的环境信息。

## 记录什么

比如：
- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好的语音
- 音箱/房间名称
- 设备昵称
- 任何与环境相关的细节

## 示例

```markdown
### 摄像头

- 客厅 → 主要区域，180° 广角
- 前门 → 入口，运动触发

### SSH

- 家庭服务器 → 192.168.1.100，用户：admin

### TTS

- 偏好语音："Nova"（温暖，略带英式）
- 默认音箱：厨房 HomePod
```

## 为什么要分开？

技能是共享的。你的设置是你的。分开存放意味着你可以在更新技能时不丢失笔记，也可以在分享技能时不泄露你的基础设施。
